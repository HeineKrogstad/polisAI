# Бэкенд для интеграции с YandexGPT

Этот сервис предоставляет API для взаимодействия с YandexGPT с использованием официального SDK Yandex Cloud.

## Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте файл `.env` в корневой директории бэкенда
2. Добавьте в него ваш API-ключ YandexGPT:

```
YANDEX_API_KEY=ваш_ключ_api
```

## Запуск

```bash
python app.py
```

Сервис будет доступен по адресу: http://localhost:5001

## API

### POST /api/chat

Отправляет сообщение в YandexGPT и возвращает ответ.

**Запрос:**

```json
{
  "message": "Ваше сообщение"
}
```

**Ответ:**

```json
{
  "answer": "Ответ от YandexGPT"
}
```

**Ошибка:**

```json
{
  "error": "Сообщение об ошибке"
}
```

## Преимущества использования SDK Yandex Cloud

1. **Типизация**: SDK предоставляет типизированные классы для запросов и ответов
2. **Безопасность**: Автоматическая обработка аутентификации
3. **Надежность**: Официальная библиотека с поддержкой от Яндекса
4. **Удобство**: Упрощенный интерфейс для работы с API
